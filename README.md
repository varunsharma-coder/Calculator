# Calculator

A java GUI calculator created using swing. Utilized arthimetics such as square root, percents, and vector operations. 

# Features 

Addition, subtraction, multiplication, division 
Percent 
Square root 
Vector addition 
Error handling division by 0 
Clear and Delete Functions 

# How to run 
Compile the files:
javac Main.java CalcPanel.java Vector.java

# Run program 
java main

Main.java
